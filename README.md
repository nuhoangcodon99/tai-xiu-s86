# Os.Core.WinVip

### Install:
- Step 1: ```npm install -g yarn```
- Step 2: ```yarn```
### Config:
- rename ```.env.example``` to ```.env``` and add require config in this
### Run debug or start: 
```yarn dev``` or ```yarn start```
